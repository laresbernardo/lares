# Left or Right N characters of a string

This functions lets the user extract the first or last n characters of a
string or vector of strings.

## Usage

``` r
left(string, n = 1)

right(string, n = 1)
```

## Arguments

- string:

  String or Vector.

- n:

  Integer. How many characters starting on right/left?

## Value

Character. Trimmed strings.

``` r

#> [1] "rdo" "res" "V"  
```
