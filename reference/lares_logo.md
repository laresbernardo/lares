# Print lares R library Logo

Used "ASCII Art Generator" from manytools.org to convert logo to ASCII.
[Visit](https://manytools.org/hacker-tools/convert-images-to-ascii-art/).

## Usage

``` r
lares_logo(version = TRUE)
```

## Arguments

- version:

  Boolean. Include R and lares version?
